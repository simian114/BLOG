# html specification 을 읽어보자.

---

## html specification 이란?

`specification` 규격 또는 사양이라고 할 수 있다. html 기능을 제공하는 제품은 이 html specification 을 보고 제품을 만들면 되는것.
그렇다고해서 무조건 따라야하는 강제성이 있는건 아니다. 그렇기 때문에 각 브라우저 엔진마다 특정 기능에 따른 동작이 다르기도하다.

---

## 그러면 왜 읽어?

많이 알고있으면 분명 언젠간 도움이 됨
